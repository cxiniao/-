# 抖音评论监控系统 - 产品介绍

## 产品概述

抖音评论监控系统是一款专业的抖音视频评论监控工具，帮助用户实时追踪指定视频的评论动态，及时发现新评论并进行数据分析。软件采用桌面客户端设计，操作便捷，运行稳定，是抖音运营、内容监测的得力助手。

---

## 核心功能

### 1. 视频评论实时监控

- **多视频同时监控**：支持添加多个抖音视频进行批量监控，单个应用最多可监控 200 个视频
- **评论变化检测**：系统每 1 秒自动检查视频评论数量变化
- **新评论实时提醒**：发现新评论时自动统计并记录，实时显示新评论数量
- **评论时间追踪**：自动记录每条评论的发布时间，便于分析评论趋势

### 2. 智能评论筛选

- **关键词筛选**：仅显示包含指定关键词的评论，快速定位目标评论
- **昵称过滤**：排除指定用户昵称的评论，过滤广告和垃圾评论
- **地区筛选**：支持按地区筛选评论，可选择排除或仅显示特定地区

### 3. 视频搜索采集

- **关键词搜索**：输入视频关键词，快速搜索抖音平台相关视频
- **多维度筛选**：
  - 发布时间：全部、今天、一周内、一月内
  - 排序方式：综合排序、最新发布、最多点赞、最多评论
  - 视频时长：全部、1分钟以内、1-5分钟、5分钟以上
  - 采集数量：可设置采集 1-500 个视频
- **封面预览**：以缩略图形式展示搜索结果，直观浏览视频内容
- **一键添加到监控**：选中视频后可快速添加到监控列表
- **批量操作支持**：支持全选、批量导入、批量删除

### 4. 视频管理功能

- **便捷添加视频**：支持输入视频链接或视频 ID 快速添加
- **批量导入**：支持从文本文件批量导入多个视频
- **列表管理**：直观展示已添加视频列表，支持删除管理

### 5. 数据本地存储

- **本地数据保存**：所有视频和评论数据保存在本地，保护数据安全
- **自动清理机制**：可设置自动清理时间（1-720 小时），自动清理过期数据
- **历史记录追溯**：可查看历史评论数据，便于分析和对比

### 6. 实时状态显示

- **监控状态直观展示**：界面实时显示监控状态、运行轮次、新评论统计
- **操作日志记录**：完整记录监控过程和系统操作，便于问题排查

## 使用说明

1. **添加视频**：在输入框中粘贴抖音视频链接或输入视频 ID，点击添加
2. **配置筛选**：根据需要设置关键词、昵称、地区等筛选条件
3. **启动监控**：点击"开始监控"按钮，系统将自动轮询监控
4. **查看结果**：实时查看新评论数据和监控统计信息
